Grid system

The grid system has been completely reworked on v2.0.

For this reason, some things may not work as expected in the LTS version (for now).

Example: Fluid bubble not work with grid system.

Less features

Many things has been added on v2.0 (ex. Alerts).

Therefore, these functions are not implemented in the LTS version.
